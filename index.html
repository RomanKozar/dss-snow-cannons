📐 Про метод ▶

Нормування вхідних даних у діапазон [0;1]. Для кожного критерію: мінімальне значення → 0, максимальне → 1 (або навпаки для «витрат»). Далі обчислюються інтегровані оцінки (згортки).

M₃(xⱼ) = Σ αᵢ · Oᵢⱼ (середня / лінійна)
M₁(xⱼ) = 1 / Σ(αᵢ/Oᵢⱼ) (песимістична)
M₂(xⱼ) = Π(Oᵢⱼ^αᵢ) (обережна / геометрична)
M₄(xⱼ) = √(Σ αᵢ·Oᵢⱼ²) (оптимістична)
🎯 Про метод 2 ▶

Метод заснований на відстані від «точки задоволення» — уявної ідеальної альтернативи T={t₁, t₂, …, tₘ}.

zᵢⱼ = 1 − |tᵢ − Oᵢⱼ| / max{tᵢ − min Oᵢⱼ; max Oᵢⱼ − tᵢ}

Потім до нормованої матриці Z застосовується одна із згорток.
